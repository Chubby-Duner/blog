input样式问题

# 当设置input的placeholder字体大小与input字体大小不一致的时候，placeholder不能垂直居中？

# 更改后

# 解决办法

# 1.通过改变position并设置top解决问题

input::placeholder {
  font-size: 0.7rem;  // 更改字体大小
  color: #aab2bd; // 更改字体颜色
  position: relative;
  top:-0.1rem%;
}

# input如何去掉输入框获取焦点时的外边框

input去掉输入框获取焦点时的外边框，只需要设置css样式为：

input {
    background: none;
    border: none;
    /* 重点是 outline: none;  */
    outline: none;
}
