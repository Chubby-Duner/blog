Set和Map

Set: 返回一个不重复的数组，可作为数组去重和去重字符串的作用

// 例一   (数组去重)
const set = new Set([1, 2, 3, 4, 4]);
[...set] // [1, 2, 3, 4]

// （去除字符串里面的重复字符）
let resStr = [...new Set('ababbc')].join('')
console.log('resStr ==> ', resStr);  // "abc"
