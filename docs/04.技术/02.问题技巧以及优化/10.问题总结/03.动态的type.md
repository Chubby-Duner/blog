::: tip 用法
（如el-tag）
使用过滤器根据数据里的状态判断
:::
```js
<el-tag :type="row.status | statusFilter">
   {{ row.status }}
</el-tag>

filters: {
  statusFilter(status) {
    const statusMap = {
      published: 'success',
      draft: 'info',
      deleted: 'danger'
    }
    return statusMap[status]
  }
},
```
